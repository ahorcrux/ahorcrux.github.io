# 下载工具

# 使用

```shell
java -jar .\jplag-5.1.0-jar-with-dependencies.jar codecheck -l c -r tmp
```

`codecheck`: root 目录

`-l`: 语言类型，参考：https://github.com/jplag/JPlag

`-r`: 输出结果文件

`-s`: 子目录路径
