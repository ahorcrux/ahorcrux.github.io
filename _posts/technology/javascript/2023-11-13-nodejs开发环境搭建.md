## 安装

安装最新版本
```shell
sudo apt-get update
sudo apt install nodejs
```

安装指定版本：如18.x

```shell
curl -sL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
```

测试安装是否成功
```shell
node --version
npm --version
```
## 编译

安装所需依赖包
```shell
npm install
```

编译
```shell
npm run build
```

## 运行

```shell
npm run serve
```

## 卸载

```shell
sudo apt-get remove nodejs
```
