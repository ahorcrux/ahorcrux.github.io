+ **screen**
  
  + 创建窗口
    
    ```shell
    screen -S
    ```
    
    例如：***screen -S demo1***
  
  + 查看已创建窗口
    
    ```shell
    screen -ls
    ```
  
  + 退出当前窗口
    
    ctrl + a ，然后按 d
  
  + 恢复窗口
    
    ```shell
    screen -r 窗口名
    ```
  
          例如：***screen -r demo1***
