## screen
  
  创建后台运行的窗口。

+ 创建窗口
  
  ```shell
  screen -S
  ```
  
  例如：***screen -S demo1***

+ 查看已创建窗口
  
  ```shell
  screen -ls
  ```

+ 退出当前窗口
  
  ctrl + a ，然后按 d

+ 恢复窗口
  
  ```shell
  screen -r 窗口名
  ```
  例如：***screen -r demo1***

## curl
  
+ GET
  
  ```shell
  curl https://proxy.mimvp.com/login?user=admin&passwd=12345678  
  ```

+ POST
  
  ```shell
  curl  http://localhost:10001 -X POST -H "Content-Type: application/json" --data '{"jsonrpc":"2.0","method":"eth_chainId","params":[],"id":1}'
  ```

## lsof

+ 查看某个端口是否开放。

  ```shell
  lsof -i:80
  ```

  ```shell
  telnet 192.168.0.1 80
  ```

+ 查看所有开放的端口
  
  ```shell
  netstat -aptn
  ```

  ```shell
  iptables -nL
  ```

+ 防火墙

  查看防火墙状态
  ```shell
  ufw status
  ```

  开启防火墙
  ```shell
  ufw enable
  ```
  开放防火墙端口
  ```shell
  ufw allow 22
  ```
  重启防火墙
  ```shell
  ufw reload
  ```
