### 运行验证者节点

1. 安装 0gchaind

    ```shell
    git clone -b testnet https://github.com/0glabs/0g-evmos.git
    ./0g-evmos/networks/testnet/install.sh
    source .profile
    ```

2. 设置 Chain ID
   
    ```shell
    0gchaind config chain-id zgtendermint_16600-1
    ```  

3. 初始化节点
   
    ```shell
    0gchaind init <your_validator_name> --chain-id zgtendermint_16600-1
    ```
    `<your_validator_name>`: 自定义验证者名称。

    默认 `home` 目录位于 `~/.0gchain` ，节点相关的重要配置项都在文件 `app.toml` 和 `config.toml` 中。
    
4. 配置 Genesis
   
    ```shell
    rm ~/.0gchain/config/genesis.json
    wget -P ~/.0gchain/config https://github.com/0glabs/0g-chain/releases/download/v0.1.0/genesis.json
    ```

    删除默认 `genesis.json` 文件，下载最新的版本并复制到 `$HOME/.0gchain/config` 目录中。
5. 配置 Seeds 节点
   
    官方教程是修改 `config.tom` 文件中的 `seeds` 和 `persistent_peers` 配置。

    ```shell
    SEEDS="c4d619f6088cb0b24b4ab43a0510bf9251ab5d7f@54.241.167.190:26656,44d11d4ba92a01b520923f51632d2450984d5886@54.176.175.48:26656,f2693dd86766b5bf8fd6ab87e2e970d564d20aff@54.193.250.204:26656,f878d40c538c8c23653a5b70f615f8dccec6fb9f@54.215.187.94:26656"
    PEERS="325c942608727d45f9cb87fb2c4b4fdd6be7e314@95.216.42.217:26656,5a202fb905f20f96d8ff0726f0c0756d17cf23d8@43.248.98.100:26656"
    sed -i "s/persistent_peers = \"\"/persistent_peers = \"$PEERS\"/" $HOME/.0gchain/config/config.toml
    sed -i "s/seeds = \"\"/seeds = \"$SEEDS\"/" $HOME/.0gchain/config/config.toml
    ```

    但是在实际操作中，上述节点信息经常连接超时或者认证失败，另外一种更简单的方式是去 [smeby.fun](https://smeby.fun/) 下载最近的 `addrbook.json` 文件，由程序自动加载节点配置：

    ```shell
    wget https://smeby.fun/0gchaind-addrbook.json -O $HOME/.0gchain/config/addrbook.json
    ```
    

6. 运行
   
    ```shell
    0gchaind start
    ```

### 创建 Validator

1. 创建账户
   
    新建账户：

    ```shell
    0gchaind keys add <key_name> --eth
    ```

    导入已有组记词，恢复账户：
    ```shell
    0gchaind keys add wallet --recover --eth
    ```

    重命令钱包：

    ```shell
    0gchaind keys rename wallet test-18-node
    ```

2. 获取测试币
   
   进入[官方水龙头](https://faucet.0g.ai/)，输入钱包私钥对应的 `evm` 地址即可

3. 创建验证者
   
    ```shell
    0gchaind tx staking create-validator \
    --amount=<staking_amount>ua0gi \
    --pubkey=$(0gchaind tendermint show-validator) \
    --moniker="<your_validator_name>" \
    --chain-id=zgtendermint_16600-1 \
    --commission-rate="0.10" \
    --commission-max-rate="0.20" \
    --commission-max-change-rate="0.01" \
    --min-self-delegation="1" \
    --from=<key_name> \
    --gas=auto \
    --gas-adjustment=1.4
    ```

    `<staking_amount>`：抵押的数量

    `<your_validator_name>`：验证者节点名称

    `<key_name>`： 钱包名称
